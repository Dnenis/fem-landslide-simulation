# Landslide FEM Simulation (VP2D)

Finite Element simulation of a slow-moving landslide 
using a viscoplastic material model (Perzyna).

## What this project does
- Models slope stability under two scenarios (stable / progressive failure)
- Uses VP2D + MeshTools2D for mesh generation and simulation
- Visualized with ParaView

## Key results
- Task 8: ~1.0m displacement, classified as "Extremely Slow"
- Task 12: Progressive failure with softening → "Rapid" velocity class

## Tools used
VP2D · MeshTools2D · Inkscape · ParaView · LaTeX
